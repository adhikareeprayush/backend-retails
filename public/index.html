Browser security (CORS)

Cross-origin requests from arbitrary websites are blocked unless their origin is listed in ALLOWED_ORIGINS (or related env vars). Server-side tools do not send an Origin header and are unaffected by CORS — use rate limits, auth, and secrets for real protection.

Backend Mart API · Same-origin apps can call this API without extra CORS configuration when hosted here.
